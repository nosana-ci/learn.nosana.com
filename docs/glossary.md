# Glossary

Key terms and definitions used throughout the Nosana documentation.

## A

### API Key
An authentication token used to access the Nosana API and SDK.

## D

### Deployment
A running instance of a workload on the Nosana network.

## J

### Job Definition
A specification that defines how a workload should be executed.

### SDK
