# Glossary

Key terms and definitions used throughout the Nosana documentation.

## A

### AI Inference
The process of using trained AI models to generate content or make predictions and decisions based on new input data.

### API Key
An authentication token used to access the Nosana API through the SDK or direct cURL requests, typically passed with each request to securely identify your account.

## C

### Community GPU market
A GPU market segment designed for new and unvalidated GPU hosts, where providers can start, test their setups, and prove reliability before moving to premium markets.

### Container
A lightweight, isolated runtime environment that packages your application code, dependencies, and system libraries. Nosana runs your workloads as containers (for example, Docker images) on GPU hosts.

### Credits
The usage-based currency required to create and run deployments on the Nosana network. Credits are consumed as your jobs run on GPU resources and can be claimed through the Nosana Dashboard.

## D

### Dashboard
The web-based control panel where you manage API keys, credits, deployments, and view the status of your workloads on Nosana.

### Deployment
A running instance of a workload on the Nosana network.

## G

### GPU Market
A pool of GPU resources with shared characteristics (such as performance and reliability) where your deployments are scheduled. Nosana offers different GPU markets, including community and premium segments.

## H

### Host
An individual GPU machine within a GPU Market that executes your jobs. Deployments are scheduled onto suitable hosts based on resource requirements.

## J

### Job
The concrete execution of your workload created from a deploymentâ€™s job definition. Jobs run your container image with the specified resources and commands and progress through states like pending, running, and completed or failed.

### Job Definition
A specification that defines how a workload should be executed.

## P

### Premium GPU market
A GPU market segment designed for validated, high-performing GPU providers that deliver reliable resources suitable for production and other critical workloads.

## S

### SDK
Software Development Kit - a collection of tools and libraries for interacting with Nosana.
